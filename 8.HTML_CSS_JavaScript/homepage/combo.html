Numpad Notation

A crash course in how to read and perform special moves in fighting games.

As you can see on the bwlow image, each direction you can press is mapped to a number on a keyboards numpad, with 5 being no direction.
So, to perform Down + Punch(P), you would press 2P.
A quarter-circle motion requires 3 inputs. Quarter circle forward is Down > Down-Forward > Forward, noted 236.
